**Goal:** Prouver qu'un vrai shell tourne dans un PTY côté hôte (`Piscine.App`), que sa sortie
s'affiche dans un terminal `xterm.js` rendu dans la webview, et que `resize` fonctionne — d'abord dans
le harnais `Piscine.DevHost` (Blazor Server, pilotable Playwright). **Critère d'acceptation #23** :
taper une commande dans le terminal in-app affiche sa sortie · resize OK · **cross-platform validé OU
repli « terminal OS » documenté**.

**Architecture:** `PtyService` (logique pure, côté hôte) vit dans **`Piscine.App`** → consommable à
l'identique par Photino *et* par le harnais Blazor Server. Le composant `Terminal.razor` (xterm.js +
JS interop) vit dans la RCL **`Piscine.Components`** (assets servis sous
`_content/Piscine.Components/...`). Le pont est câblé dans une **page DevHost** `@page "/terminal"` (de
la RCL). Aucune ligne du moteur (`Core`/`Grading`/`Git`), de `Piscine.Cli` ni de `release.yml`.

**Tech Stack (PINNED):**
- **`Porta.Pty` 1.0.7** (NuGet, 2026-01-22) — ConPTY (Win10 1809+) + shim natif `forkpty` par RID
  (linux-x64/arm64, osx-x64/arm64). API : `PtyProvider.SpawnAsync(PtyOptions, ct) → IPtyConnection`

## ⚠️ Note de risque (SPIKE) — ce qui peut forcer le repli, et le point de décision

Ce sprint a **un** risque dominant : le PTY cross-platform. Trois bascules possibles vers la
**solution de repli documentée** (« bouton *Ouvrir un terminal OS ici* + coaching d'état » — spec §5,
écarté du spike mais conçu agnostique au shell) :

1. **Restauration NuGet net10.** `Porta.Pty` cible `netstandard2.0` ; sous `TreatWarningsAsErrors`
   guetter **NU1701** (paquet restauré via netstandard) et **NU1605** (downgrade transitif). S'il faut
   un `<NoWarn>NU1701</NoWarn>` ciblé *uniquement sur ce paquet*, l'assumer dans le `.csproj` de
   `Piscine.App` (documenté inline). Échec de restauration dur → repli.
2. **Spawn natif.** Le shim `libporta_pty` peut manquer pour un RID, ou ConPTY peut ne pas être dispo
   (éditions Windows **N** sans WebView2/MediaFeaturePack — pertinent pour S9). Si `SpawnAsync` jette
   sur la plateforme de dev (Windows ici), **STOP** → documenter + repli.
3. **Débit / backpressure SignalR** (DevHost) : si pousser stdout vers `xterm.write` sature le circuit
   (`dir`, `ls -R`), la coalescence (T3) doit suffire ; sinon, noter la limite (le vrai hôte cible est
   Photino, pas SignalR — relativiser).

**POINT DE DÉCISION (fin T1).** Après `PtyService` + son test unitaire : si le PTY **spawn + echo
round-trip** passe sur Windows, on continue T2→T6. Sinon, on s'arrête, on bascule le sprint sur la
**branche repli** (documenter dans `docs/superpowers/` + composant « terminal OS ») et on clôt #23 en
« risque levé : PTY KO sur <OS>, repli adopté ». **Un agent ne peut PAS valider Linux/macOS ici** :
« cross-platform validé » au sens #23 = (i) Windows prouvé bout-en-bout par l'agent + (ii) une
**checklist smoke par OS** rédigée (T5) à exécuter par le proprio, comme pour le smoke Photino S1.

- [ ] **Step 4 — Implémentation + factory** `src/Piscine.App/Terminal/PtyService.cs`. `PtyService` est
  le service DI (sans état) ; il `StartAsync` une `PtySession` qui boucle sur `ReaderStream`.

```csharp
using System.Text;
using Porta.Pty;

namespace Piscine.App.Terminal;

/// <summary>Lance des sessions PTY (un vrai shell OS). Enregistré en DI, sans état partagé.</summary>
public sealed class PtyService
{
    public async Task<IPtySession> StartAsync(PtyStartInfo info, CancellationToken ct = default)
    {
        var options = new PtyOptions
        {
            Name = "piscine-term",
            App = info.Shell ?? PtyStartInfo.DefaultShell(),
            Cwd = info.WorkingDirectory,
            Cols = info.Cols,
            Rows = info.Rows,
        };
        var connection = await PtyProvider.SpawnAsync(options, ct);
        return new PtySession(connection);
    }

    private sealed class PtySession : IPtySession
    {
        private readonly IPtyConnection _conn;
        private readonly CancellationTokenSource _cts = new();

        public event Action<byte[]>? Output;
        public event Action<int>? Exited;

        public PtySession(IPtyConnection conn)
        {
            _conn = conn;
            _conn.ProcessExited += (_, e) => Exited?.Invoke(e.ExitCode);
            _ = PumpAsync(_cts.Token);
        }

        private async Task PumpAsync(CancellationToken ct)
        {
            var buffer = new byte[4096];
            try
            {
                while (!ct.IsCancellationRequested)
                {
                    var n = await _conn.ReaderStream.ReadAsync(buffer.AsMemory(0, buffer.Length), ct);
                    if (n <= 0) break;
                    var chunk = new byte[n];
                    Array.Copy(buffer, chunk, n);
                    Output?.Invoke(chunk);
                }
            }
            catch (OperationCanceledException) { /* arrêt normal */ }
            catch (IOException) { /* PTY fermé */ }
        }

        public async Task WriteAsync(string data, CancellationToken ct = default)
        {
            var bytes = Encoding.UTF8.GetBytes(data);
            await _conn.WriterStream.WriteAsync(bytes.AsMemory(), ct);
            await _conn.WriterStream.FlushAsync(ct);
        }

        public void Resize(int cols, int rows) => _conn.Resize(cols, rows);

        public async ValueTask DisposeAsync()
        {
            await _cts.CancelAsync();
            try { _conn.Kill(); } catch { /* déjà mort */ }
            _conn.Dispose();
            _cts.Dispose();
        }
    }
}
```

```razor
@namespace Piscine.Components.Components.Terminal
@implements IAsyncDisposable
@inject IJSRuntime JS

<div @ref="_host" class="piscine-terminal"></div>

@code {
    private ElementReference _host;
    private IJSObjectReference? _module;
    private IJSObjectReference? _term;       // handle JS du terminal xterm
    private DotNetObjectReference<Terminal>? _self;

    /// <summary>Invoqué quand la recrue tape (xterm onData) — relayé vers le PTY par le parent.</summary>
    [Parameter] public EventCallback<string> OnData { get; set; }
    /// <summary>Invoqué au resize (cols,rows) après fit — relayé vers PtyService.Resize.</summary>
    [Parameter] public EventCallback<(int Cols, int Rows)> OnResize { get; set; }

    protected override async Task OnAfterRenderAsync(bool firstRender)
    {
        if (!firstRender) return;
        _self = DotNetObjectReference.Create(this);
        _module = await JS.InvokeAsync<IJSObjectReference>(
            "import",
            "./_content/Piscine.Components/Components/Terminal/Terminal.razor.js");
        _term = await _module.InvokeAsync<IJSObjectReference>("createTerminal", _host, _self);
    }

    /// <summary>Pousse des octets PTY (base64) vers xterm.write.</summary>
    public async Task WriteAsync(string base64) =>
        await (_module?.InvokeVoidAsync("write", _term, base64) ?? ValueTask.CompletedTask);

    [JSInvokable] public Task HandleData(string data) => OnData.InvokeAsync(data);
    [JSInvokable] public Task HandleResize(int cols, int rows) => OnResize.InvokeAsync((cols, rows));

    public async ValueTask DisposeAsync()
    {
        if (_module is not null && _term is not null)
            try { await _module.InvokeVoidAsync("dispose", _term); } catch { /* circuit mort */ }
        if (_term is not null) await _term.DisposeAsync();
        if (_module is not null) await _module.DisposeAsync();
        _self?.Dispose();
    }
}
```

- [ ] **Step 3 — Le module ESM colocalisé** `Components/Terminal/Terminal.razor.js`. Il charge les
  assets vendorisés (UMD → globals), crée le terminal, branche `onData`/`onResize` vers .NET, et
  expose `write` (décodage base64 → `Uint8Array` → `term.write`).

```javascript
// Charge xterm + fit addon vendorisés dans la RCL (préfixe _content obligatoire).
async function ensureLib() {
    if (window.Terminal && window.FitAddon) return;
    await loadScript("/_content/Piscine.Components/lib/xterm/xterm.js");
    await loadScript("/_content/Piscine.Components/lib/xterm/addon-fit.js");
}
function loadScript(src) {
    return new Promise((res, rej) => {
        const s = document.createElement("script");
        s.src = src; s.onload = res; s.onerror = rej;
        document.head.appendChild(s);
    });
}

export async function createTerminal(host, dotnet) {
    await ensureLib();
    const term = new window.Terminal({ convertEol: true, fontSize: 14 });
    const fit = new window.FitAddon.FitAddon();
    term.loadAddon(fit);
    term.open(host);
    fit.fit();

    term.onData(d => dotnet.invokeMethodAsync("HandleData", d));

    const pushResize = () => {
        fit.fit();
        dotnet.invokeMethodAsync("HandleResize", term.cols, term.rows);
    };
    const ro = new ResizeObserver(pushResize);
    ro.observe(host);
    pushResize(); // dimensions initiales

    return { term, fit, ro };
}

export function write(handle, base64) {
    const bin = atob(base64);
    const bytes = new Uint8Array(bin.length);
    for (let i = 0; i < bin.length; i++) bytes[i] = bin.charCodeAt(i);
    handle.term.write(bytes);
}

export function dispose(handle) {
    try { handle.ro.disconnect(); } catch {}
    try { handle.term.dispose(); } catch {}
}
```

```razor
    private Terminal? _terminal;

    protected override async Task OnAfterRenderAsync(bool firstRender)
    {
```

```csharp
        using var pw = await Playwright.CreateAsync();
        IBrowser browser;
        try { browser = await pw.Chromium.LaunchAsync(); }
        catch (PlaywrightException) { return; } // skip propre

        await using (browser)
        {
            var page = await browser.NewPageAsync();
            await page.GotoAsync($"{BaseUrl}/terminal", new() { Timeout = 30_000 });
            await page.WaitForSelectorAsync(".xterm", new() { Timeout = 30_000 });
            await page.ClickAsync(".xterm");                 // focus
            await page.Keyboard.TypeAsync("echo PISCINE_E2E");
            await page.Keyboard.PressAsync("Enter");
            // La sortie écho de la commande apparaît dans le buffer xterm rendu.
            await page.WaitForFunctionAsync(
                "() => document.querySelector('.xterm-screen')?.textContent?.includes('PISCINE_E2E')",
                new PageWaitForFunctionOptions { Timeout = 30_000 });
        }
    }

    private static async Task WaitForServerAsync(TimeSpan timeout)
    {
        using var http = new HttpClient { Timeout = TimeSpan.FromSeconds(5) };
        var deadline = DateTime.UtcNow + timeout;
        while (DateTime.UtcNow < deadline)
        {
            try { if ((await http.GetAsync(BaseUrl)).IsSuccessStatusCode) return; }
            catch (HttpRequestException) { } catch (TaskCanceledException) { }
            await Task.Delay(500);
        }
        throw new TimeoutException($"DevHost muet sur {BaseUrl} ({timeout.TotalSeconds:0}s).");
    }

    private static string FindRepoRoot()
    {
        var dir = new DirectoryInfo(AppContext.BaseDirectory);
        while (dir is not null && !File.Exists(Path.Combine(dir.FullName, "Piscine.slnx"))) dir = dir.Parent;
        return dir?.FullName ?? throw new InvalidOperationException("Piscine.slnx introuvable.");
    }
}
```
