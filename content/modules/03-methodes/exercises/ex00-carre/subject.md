# ex00-carre — Carré

## Objectif

Lis **un entier** `n` sur l'entrée standard, puis affiche son **carré** (`n × n`).

Le but est d'écrire une **méthode** `Carre` qui prend un `int` et **renvoie** son carré, puis de
l'appeler.

Exemples : `5` → `25` · `0` → `0` · `12` → `144`.

## Livrable

- `Carre.cs`

## Indices

- Déclare la méthode : `static int Carre(int n) => n * n;` (place-la en bas du fichier).
- Affiche le résultat : `System.Console.WriteLine(Carre(n));`.
