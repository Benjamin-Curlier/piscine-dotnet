# ex00-compte-rebours — Compte à rebours

## Objectif

Lis **un entier** `N` sur l'entrée standard, puis affiche les nombres de `N` à `1`,
**un par ligne**.

Exemple : `3` →
```
3
2
1
```

## Livrable

- `CompteRebours.cs`

## Indices

- Une boucle `for` qui **décroît** : `for (var i = n; i >= 1; i--)`.
- Affiche `i` à chaque tour avec `System.Console.WriteLine(i)`.
