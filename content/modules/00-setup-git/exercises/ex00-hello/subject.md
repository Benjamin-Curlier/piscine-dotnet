# ex00-hello — Hello, Piscine

## Objectif

Écris un programme qui affiche **exactement** :

```
Hello, Piscine!
```

(avec un retour à la ligne final).

## Livrable

- `Hello.cs`

## Indices

- `System.Console.WriteLine("...")` affiche un texte suivi d'un retour à la ligne.
- Respecte la casse, la virgule, l'espace et le `!` au caractère près.

## Pour rendre

```bash
piscine check ex00-hello       # vérifie en local
git add . && git commit -m "ex00-hello" && git push origin main
```
